### 顺序查找

+ 思路

```
从数组的开头到结尾,顺序挨个比较查找。
```

+ 算法复杂度

&ensp;&ensp;&ensp;&ensp;平均算法复杂度为：O(n)= n; 最坏情况的算法复杂度为：O(n) = n;

#### cpp实现

```cpp
int orderSearch(int dest_arr[],int  dest) {

    int arr_len = sizeof(dest_arr)/sizeof(*dest_arr);
    for(int i = 0; i <= arr_len; i++) {
        if (dest_arr[i] == dest) {
            return i;
        }
    }
    return -1;
}
```
